# didaci-canvas-ar

Augmented Reality web app built with A-Frame and AR.js.
Points a camera at a QR marker and renders an interactive 3D model on top of it.
Built for real events — includes coupon reward system.

---

## How it works

1. Open the web on mobile
2. Allow camera access
3. Point at the QR marker
4. A 3D model appears — tap it to trigger animations
5. Collect your coupon

## Features

- QR marker detection via AR.js
- Interactive 3D models (dog + cat) with animations
- Jump, spin and dance animations on tap
- Coupon reward system
- Mobile optimized
